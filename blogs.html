Some Questions And Answers :

Question:1) What is the difference among var , const and let?

Answer: The scope of a var variable is functional scope.It can be updated and re-declared into the scope.It can be updated and re-declared into the scope.
The scope of a const variable is block scope.It cannot be updated or re-declared into the scope. It cannot be declared without initialization.
The scope of a let variable is block scope.It can be updated but cannot be re-declared into the scope. It can be declared without initialization.



Question:2) What is the difference between Regular function and Arrow function?

Answer : Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword. However, the arrow functions are only callable and not constructible,arrow functions can never be used as constructor functions.Arrow functions are best for callbacks or methods like map,reduce or forEach .Arrow functions are best for callbacks or methods like map , reduce , or forEach .
